# Project 04: ONC Media Inventory Manager
Part of the 30-Day Python & FastAPI Sprint.

## 📌 Overview
A specialized inventory management system for **Overcomers Nation Church** to track media gear (cameras, mics, and lighting).

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **Data Structure:** Dictionaries & Classes
- **Version Control:** Git/GitHub

## 🚀 Features
- Add new media equipment.
- Update equipment status (Operational/Maintenance).
- Search for specific gear.
